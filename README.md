# Nashville Housing Data Cleaning & Analysis Project

## Project Overview
An end-to-end data cleaning and housing market analysis project using MySQL, Python (Pandas), and Tableau Public.

The project focuses on transforming raw Nashville housing data into a clean and analytics-ready dataset, followed by exploratory data analysis and interactive visualizations.
---
## Tools & Technologies
- MySQL
- SQL
- Python
- Pandas
- Tableau Public
---
## Project Objectives
- Clean and standardize raw housing data
- Handle missing values and duplicate records
- Split and normalize address fields
- Perform exploratory data analysis using Python
- Build interactive Tableau dashboards for housing insights
---
## SQL Data Cleaning Techniques Used
- Data Standardization
- Self Joins
- Handling NULL Values
- Address Splitting
- CASE Statements
- Window Functions
- Duplicate Removal
- Data Transformation
---
## Python Analysis
Performed exploratory data analysis using Pandas:
- Dataset statistics
- Missing value analysis
- Average sale price analysis
- Land use distribution
- Property distribution by city
---
## Tableau Dashboard Features
- Housing KPI Summary
- Property Distribution by City
- Land Use Distribution
- Housing Price vs Land Value Analysis
---
## Key Insights
- Average property sale price exceeded $252K
- Single-family homes dominated the housing market
- Goodlettsville had the highest property concentration
- Significant missing-value trends identified in owner and property data
